Knowledge Base Prolog
=====================

La cartella ``kb`` contiene la base di conoscenza Prolog usata dal progetto per
fornire un livello di ragionamento esplicito sul tipo di allenamento, sul livello
di intensita' e sulla durata ottimale.

File principali
---------------

``kb/kb.pl``
   Contiene fatti e regole Prolog, tra cui ``recommended_workout/3``,
   ``recommended_intensity/4`` e ``optimal_duration/4``.

``kb/doc/index.html``
   Documentazione HTML generata per consultare i predicati della Knowledge Base.
